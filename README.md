## Usage

**Shows usage text, as defined in `lib/help.txt`**

```
http://localhost:8080
```

**github.com with CORS headers**

```
http://localhost:8080/http://github.com
```

**Proxies `https://google.com`**

```
http://localhost:8080/google.com:443
```
